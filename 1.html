HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applucations.

HTML not only represent structural meanings for the body, such as titles, paragraphs, lists, etc.,


but also provide a way to create structured documents with links, citations, and other items. It can then be used to embedde images and objects and create interactive forms.

HTML is written in the form of an HTML element witha "tag" enclosed in an bracket in the content of a web page. HTML can include or loadjavascript, which affects the behavior of HTML processing devices such as webbrowsers, and scripts such as CSS that define the appearance and placement of the body and other items.

W3C,co-director of HTML and CSS standards, recommends the use of CSS for clear and display markup.
